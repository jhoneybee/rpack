# 更新日志 

遵循 Semantic Versioning 2.0.0 语义化版本规范。

## 最新版本

| 描述               | 版本号
|----                |------

## 发布周期

 - 主版本号：含有破坏性更新和新特性，不在发布周期内。
 - 次版本号：每月发布一个带有新特性的向下兼容的版本。
 - 修订版本号：每周会进行日常 bugfix 更新。（如果有紧急的 bugfix，则任何时候都可发布）
